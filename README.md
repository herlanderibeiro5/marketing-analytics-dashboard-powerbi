# Marketing Analytics Dashboard | Power BI + SQL Server

Interactive executive dashboard for campaign performance, lead generation and conversion analytics.

## Summary

This project presents a complete Marketing Analytics solution developed using Power BI and SQL Server.

The dashboard provides an executive overview of campaign performance, revenue growth, lead generation, conversion rates and ROI metrics through interactive visualizations and advanced DAX calculations.

The project includes dimensional data modeling, KPI analysis, time intelligence functions and business storytelling techniques to support data-driven decision making.

## Dashboard Overview

![Dashboard](assets/images/dashboard-overwiew.png)

The dashboard was designed to analyze marketing campaign performance through:

- Revenue and investment tracking
- Lead generation and conversion monitoring
- Campaign conversion efficiency
- Revenue growth trend analysis

## Main Features

- Interactive Power BI dashboard
- KPI cards with trend comparison
- Revenue trend analysis
- Campaign conversion funnel
- Dynamic slicers and filtering
- Time intelligence calculations
- Conversion rate analysis
- ROI performance tracking
- Executive business insights
- Dimensional data modeling
- ROI performance indicators
- Interactive filtering by date, campaign, channel and city
